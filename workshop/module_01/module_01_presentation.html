THE USE CASE

4,591 borrowers. One question: will they pay us back?

Every lender faces the same decision. An applicant walks in with a slim file, and you have minutes to commit money or walk away. Traditional credit scoring has not moved meaningfully since the 1980s.

For Module 01 we start with four numeric features — age, employment years, distance from branch, prior lenders. Module 03 adds the categorical columns and the joins.

Step 01 · Authenticate.

One call. Every estimator in the session picks up the connection.
module_01_your_first_nexus_model.ipynb · cell 7 
# FUNDAMENTAL_API_KEY lives in the shell env, never in a committed cell.
from fundamental import Fundamental, NEXUSClassifier, set_client

set_client(Fundamental())

print("Authentication: OK")
WHY THIS MATTERS

Configure once. Use everywhere.

set_client() registers the connection for the whole Python session. From here on, every NEXUSClassifier or NEXUSRegressor you instantiate reaches the same account — no passing client objects around, no rediscovering the key on every call.

Step 02 · Train.

Numeric features only. Quality mode. A few minutes of wall clock.
module_01_your_first_nexus_model.ipynb · cells 16 & 19 
FEATURES = ["age", "distance_from_branch_miles",
            "num_previous_lenders", "total_employment_years"]

X_train = train_df[FEATURES]
y_train = train_df["default_flag"]

nexus = NEXUSClassifier(mode="quality")
nexus.fit(X_train, y_train)

MODEL_ID = nexus.trained_model_id_
print(MODEL_ID)  # e.g. 72d000e3-9216-44ba-...
WHY THIS MATTERS

The id outlives the session.

trained_model_id_ lives in Fundamental's registry. Store it. Every module from 02 onward reloads the weights from that string — no retraining, no local checkpoints, no mystery about which model is which.

mode="quality" runs longer, trains to a better AUC. mode="speed" returns faster — for iteration. Module 02 compares them on the same data.

11 / 15

Step 03 · Score the holdout.

Standard sklearn metrics. One number to report back.
module_01_your_first_nexus_model.ipynb · cells 23 & 25 
from sklearn.metrics import classification_report, roc_auc_score

X_holdout = holdout_df[FEATURES]
y_holdout = holdout_df["default_flag"]

y_pred = nexus.predict(X_holdout)

print(classification_report(y_holdout, y_pred,
    target_names=["No default", "Default"]))

auc = roc_auc_score(y_holdout, y_pred)
print(f"ROC AUC: {auc:.4f}")
WHY THIS MATTERS

Three metrics. Three questions.

Precision asks, of borrowers we flagged, how many really defaulted — false alarms. Recall asks, of real defaulters, how many we caught — missed defaults. AUC is the single number for overall discriminative power. Module 02 tunes the threshold between them.

Confusion matrix, in business terms: TN correctly approved a safe borrower. FP flagged a safe borrower (false alarm). FN missed a defaulter (the expensive one). TP caught a defaulter.
